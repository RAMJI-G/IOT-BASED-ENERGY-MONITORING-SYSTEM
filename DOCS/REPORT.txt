# 2. Aim of the Project

The aim of this project is to design and develop an ESP32-based IoT Smart Energy Monitoring and Home Automation System capable of monitoring electrical parameters such as voltage, current, and power while enabling wireless appliance control using Blynk IoT and Google Assistant through Sinric Pro.

---

# 3. Abstract

This project presents an AI-enabled IoT Smart Energy Monitoring and Home Automation System using ESP32. The system integrates energy monitoring, relay-based automation, wireless communication, and cloud IoT technology into a single embedded platform.

The proposed system measures electrical parameters including voltage, current, and power using ACS712 and ZMPT101B sensors. The measured data is displayed on an OLED display and transmitted to the Blynk IoT cloud platform for remote monitoring and analytics.

The system also supports smart appliance automation using relay modules to control devices such as bulbs and fans. Google Assistant voice control is implemented through Sinric Pro, enabling users to operate appliances using voice commands.

This project combines embedded systems, IoT, automation, wireless communication, cloud monitoring, and smart home technologies into a professional engineering solution suitable for smart homes, industries, and energy-saving applications.

# 6. Core Concept

The ESP32 acts as the central controller of the system.

The ACS712 current sensor and ZMPT101B voltage sensor measure electrical parameters from connected loads.

The ESP32 processes sensor data and calculates:

* Voltage
* Current
* Power

The calculated values are:

* Displayed on OLED
* Sent to Blynk Cloud
* Visualized using SuperChart

Relay modules are used to control appliances such as bulbs and fans.

Sinric Pro connects the ESP32 with Google Assistant for voice-based automation.

# 8. Complete Wiring and Configuration

## OLED Connections

| OLED Pin | ESP32 Pin |
| -------- | --------- |
| VCC      | 3.3V      |
| GND      | GND       |
| SDA      | GPIO21    |
| SCL      | GPIO22    |

## ACS712 Connections

| ACS712 Pin | ESP32 Pin |
| ---------- | --------- |
| VCC        | 5V        |
| GND        | GND       |
| OUT        | GPIO34    |

## ZMPT101B Connections

| ZMPT101B Pin | ESP32 Pin |
| ------------ | --------- |
| VCC          | 5V        |
| GND          | GND       |
| OUT          | GPIO35    |

## Relay Connections

| Relay Pin | ESP32 Pin |
| --------- | --------- |
| IN1       | GPIO26    |
| IN2       | GPIO27    |
| VCC       | 5V        |
| GND       | GND       |

## Appliance Connections

* Relay Channel 1 → Bulb
* Relay Channel 2 → Fan

# 10. Circuit Diagram Explanation

The ESP32 acts as the central controller connected to sensors, relays, OLED, and IoT platforms.

The ACS712 measures current flow through the load.

The ZMPT101B measures AC voltage.

The relay module controls AC appliances.

The OLED display shows real-time values.

The ESP32 connects to WiFi and uploads sensor data to Blynk cloud.

Google Assistant commands are received via Sinric Pro and processed by ESP32.

---

# 11. Working Principle

1. ESP32 powers ON.
2. ESP32 connects to WiFi.
3. Sensors continuously monitor electrical parameters.
4. ESP32 calculates power.
5. OLED displays real-time values.
6. Data is uploaded to Blynk.
7. User controls appliances using:

   * Blynk App
   * Google Assistant
8. Relay switches ON/OFF appliances.

# 14. Blynk Setup Guide

## Steps

1. Install Blynk IoT app
2. Create template
3. Create datastreams:

| Datastream | Virtual Pin |
| ---------- | ----------- |
| Voltage    | V0          |

| Fan Relay  | V4          |

* Gauge
* Switch
* SuperChart
* Label

5. Copy:

* Template ID
* Device Name
* Auth Token

6. Paste into Arduino code.
